# 聊天记录

本文件夹用于保存与 AI 助手（灵码/Lingma）的对话记录。

## 目录结构

```
chat-records/
├── README.md              # 本说明文件
├── conversation-YYYY-MM-DD.md  # 每日对话记录（自动创建）
└── ...
```

## 使用说明

1. **自动记录**：每次对话后，AI 助手会帮助整理并保存到对应的日期文件中
2. **文件命名**：按日期自动命名，格式为 `conversation-YYYY-MM-DD.md`
3. **内容格式**：包含问题、解决方案、代码修改等完整对话内容
4. **搜索查阅**：可按日期或内容关键词快速查找历史记录

## 记录规范

- 每个对话会话保存为一个独立的 Markdown 文件
- 包含时间戳、问题描述、解决方案等关键信息
- 代码修改会标注文件路径和修改内容
- 重要的经验教训会特别标注
