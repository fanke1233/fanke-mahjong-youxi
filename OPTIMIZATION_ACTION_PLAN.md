## ⚠️ **重要提醒**

### 构建注意事项

1. **构建时间增加**
   - 启用Zip Compress后，构建时间会增加2-3分钟
   - 大型项目可能需要10-15分钟
   - **请耐心等待，不要中断构建**

2. **构建失败处理**
   - 如果构建卡住，关闭Cocos Creator
   - 删除 `temp/` 和 `library/` 目录
   - 重新打开并构建

3. **验证构建产物**
   - 构建完成后，务必检查文件大小
   - 如果文件仍然很大，说明Zip Compress未生效
   - 重新检查构建设置

### 部署注意事项

1. **GitHub Pages更新延迟**
   - 推送代码后，等待2-3分钟
   - GitHub Pages需要时间重新构建站点

2. **浏览器缓存**
   - 测试时使用 `Ctrl + F5` 强制刷新
   - 或清除浏览器缓存

3. **CDN缓存**
   - 如果使用CDN，可能需要刷新CDN缓存
   - GitHub Pages自带CDN，自动更新
